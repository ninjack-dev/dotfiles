# Neovim Config Skill

Colocated script `nvim-help` works exactly like `:help <topic>` inside Neovim, but outputs plain text to stdout.

```sh
./nvim-help '<topic>'
```

Examples:

```
nvim-help 'shiftwidth'
nvim-help 'options.txt'
nvim-help MiniPairs
nvim-help 'i_CTRL-V'
```
